* ES7 (EcmaScript 2016)

年份和版本号不要搞混了，7是指第七个版本，2016是release的年份，就像 ES6 也叫 ES2015 一样

ES7 很少被提及，是因为它的 feature 太少了，而且 +没什么卵用+
